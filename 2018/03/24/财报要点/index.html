财报要点

发表于 2018-03-24 | 分类于 股海沉浮
字数统计: 603 | 阅读时长 ≈ 2

应收款

当一家公司连续两年的应收款上升同谋超过营业收入上升的幅度时，那么这家公司就没有赚多少钱，只是开出了很多白条。

  1. 卖出的货物没有收回相应的货款，令公司可利用的现金减少，影响资金的周转率；
  2. 应收款中很可能会有一部分收不回成为坏账，令公司的财务状况雪上加霜；
  3. 催收账款也会产生额外的支出，应收账款越多，这部分的支出就越大。

存货

当一家公司连续两年存货上升幅度连续两年超过营业收入上升幅度，那么这家公司的运营出现了很大的问题，生产的商品卖不出去，只能堆存仓库。

  1. 存货的增加超过了营业收入的增涨幅度，说明公司的可利用资金被越来越多的转化为了库存，令公司的可利用资金减少了；
  2. 存货卖不出去会不断地贬值；
  3. 存货随着时间地推移会变次品，或破损或报废；
  4. 存货多了需要占用更多的空间，需要更多的人力与物力管理成本增加。

负债

当一家公司的流动负债远远大于流动资产，那和这家公司已经非常接近于破产。

当出现资不抵债的状况时，公司就会面临破产。

经营活动现金流

当一家公司的经营活动现金流远远大于净利润，这样的公司就可能 在隐蔽的赢利能力。

净现金流

净现金流这个值才是真正衡量公司的收入，用来衡量公司的赢利能力理准确，理难造假。

净现金流 = 经营现金流-资本开支
公司的业绩可能会在存货和应收账款等地方被做手脚，令公司资产的数据里包含许多的水分；而净现金注牟计算中排除了存货和应收账款的误导。净现金流是公司衬衫在可以掌控、可以运用的资金，这些资金可以实际运用在公司的经营活动中。净现金流越多，公司的经营灵活度就越高，其盈利能力就越高。
