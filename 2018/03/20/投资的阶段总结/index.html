投资的阶段总结

发表于 2018-03-20 | 分类于 股海沉浮
字数统计: 793 | 阅读时长 ≈ 3
  1. 第一阶段关注大牛的公众号，并且在看到有大牛提出的股票后就毫不犹豫地买在，在这个阶段反而还是能够有盈利的，但是因为没有经过自己研究的情况下，当股票下跌时，就拿不住了，总是在想着别人卖了没有，这个股票有没有优势，还能不能拿。天天都去刷新大牛的公众号看他有没有发最新的文章，如果发表了有利的言论就信心十足，如果有不好的消息就患得患失。最后的结果就是好的股票拿不住了。

  2. 在第二阶段看了一些书，特别是一些技术类型的书之后，在每天下班之后就是研究K线图，并且根据港资大买的股票作为研究的对象，每当看到有一支股票的k线形态符合书中的描述，就有一种感觉认为这只股票一定会涨的。接着内心会涌现出一种不可抑制的贪婪，在这种贪婪的驱动下作出了冲动的决定买了第一只让我大亏的一只股票，亏损达到了20%。这只股票这我认识到了

    就算受到资金的吹捧，股票也不是一到书中所说的那个点位就会马上飞起来的。如果不是当时的市场，行业，公司的经营状况，与成长的确定都能够达标的情况下，一定不会出现一个超级强势股，要不然就算是涨最多也是题材股的爆发罢了。
    这支股票是水井坊 同时也让我认识到在市场中并不是很认为你是对的，只要你相信与坚守就一定是对的，但事实并不是如此
    当你所做的决定与和预期中的不符合时，就是你看错了，并没有任何理由，也不要对自己手中的股票抱有特殊地感情。一旦认识到错误就要毫不犹豫地止损。

  3. 第三个阶段就是听消息，当在哪里看到有人评论哪支股票的形态很好，就去研究那支股票，此时已经学到了一些东西了，也掌握了一些工具了，懂得了利用清单去对照。但是还是没有研究过基本面的情况下，就仓促的买进，到了最后因为股价没有按照预期的形态来走，就做了止损的决定。也没有再出现很大的亏损。

  4. 第四阶段慢慢地开始去了解股票的本质是什么，对于什么算低估，什么算高估有自己的判断，也更加有耐心了，能够持有现金，耐心等待买入时机的出现。同时也能够在自己持有股票在下跌的过程，因为对股票的耐心而不断加仓。而能够不断加仓的底气则来自对公司的确定性，成长性有了一定的认识。
