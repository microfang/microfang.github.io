衡量公司的三种工具

发表于 2018-03-26 | 分类于 股海沉浮
字数统计: 273 | 阅读时长 ≈ 1

公司的盈利能力

  • 公司的毛利率。通过了解公司的毛利率可以了解到公司生产的产品是否生产高附加值的产品。并且通过高利率可以排除公司的利润是否通过收购或是政府的补贴而产生的；
  • 公司的净利润率。有高的净利润率，说明公司有较强的盈利能力；
  • ROE 净资产收益率=净利润/股东权益；
  • ROA 总资产收益率 = 净利润/总资产；

公司的负债情况

  • 资产负债比。资产负债率越少越好。
  • 流动比
  • 速动比

公司的运营效率

  • 存货周转率=营业成本/平均存货 。这个数据越大说明这家公司的效率就越高；
  • 资产周转率=营业收入/总资产平均余额 数据越大说明公司的资金利用率越高，公司的运转效率也就越高。
  • 三费占比=（销售费用+管理费用+财务费用）/营业收入 可以衡量公司的成本控制能力。

财报要点

发表于 2018-03-24 | 分类于 股海沉浮
字数统计: 603 | 阅读时长 ≈ 2

应收款

当一家公司连续两年的应收款上升同谋超过营业收入上升的幅度时，那么这家公司就没有赚多少钱，只是开出了很多白条。

  1. 卖出的货物没有收回相应的货款，令公司可利用的现金减少，影响资金的周转率；
  2. 应收款中很可能会有一部分收不回成为坏账，令公司的财务状况雪上加霜；
  3. 催收账款也会产生额外的支出，应收账款越多，这部分的支出就越大。

存货

当一家公司连续两年存货上升幅度连续两年超过营业收入上升幅度，那么这家公司的运营出现了很大的问题，生产的商品卖不出去，只能堆存仓库。

  1. 存货的增加超过了营业收入的增涨幅度，说明公司的可利用资金被越来越多的转化为了库存，令公司的可利用资金减少了；
  2. 存货卖不出去会不断地贬值；
  3. 存货随着时间地推移会变次品，或破损或报废；
  4. 存货多了需要占用更多的空间，需要更多的人力与物力管理成本增加。

负债

当一家公司的流动负债远远大于流动资产，那和这家公司已经非常接近于破产。

当出现资不抵债的状况时，公司就会面临破产。

经营活动现金流

当一家公司的经营活动现金流远远大于净利润，这样的公司就可能 在隐蔽的赢利能力。

净现金流

净现金流这个值才是真正衡量公司的收入，用来衡量公司的赢利能力理准确，理难造假。

净现金流 = 经营现金流-资本开支
公司的业绩可能会在存货和应收账款等地方被做手脚，令公司资产的数据里包含许多的水分；而净现金注牟计算中排除了存货和应收账款的误导。净现金流是公司衬衫在可以掌控、可以运用的资金，这些资金可以实际运用在公司的经营活动中。净现金流越多，公司的经营灵活度就越高，其盈利能力就越高。

投资的阶段总结

发表于 2018-03-20 | 分类于 股海沉浮
字数统计: 793 | 阅读时长 ≈ 3
  1. 第一阶段关注大牛的公众号，并且在看到有大牛提出的股票后就毫不犹豫地买在，在这个阶段反而还是能够有盈利的，但是因为没有经过自己研究的情况下，当股票下跌时，就拿不住了，总是在想着别人卖了没有，这个股票有没有优势，还能不能拿。天天都去刷新大牛的公众号看他有没有发最新的文章，如果发表了有利的言论就信心十足，如果有不好的消息就患得患失。最后的结果就是好的股票拿不住了。

  2. 在第二阶段看了一些书，特别是一些技术类型的书之后，在每天下班之后就是研究K线图，并且根据港资大买的股票作为研究的对象，每当看到有一支股票的k线形态符合书中的描述，就有一种感觉认为这只股票一定会涨的。接着内心会涌现出一种不可抑制的贪婪，在这种贪婪的驱动下作出了冲动的决定买了第一只让我大亏的一只股票，亏损达到了20%。这只股票这我认识到了

    就算受到资金的吹捧，股票也不是一到书中所说的那个点位就会马上飞起来的。如果不是当时的市场，行业，公司的经营状况，与成长的确定都能够达标的情况下，一定不会出现一个超级强势股，要不然就算是涨最多也是题材股的爆发罢了。
    这支股票是水井坊 同时也让我认识到在市场中并不是很认为你是对的，只要你相信与坚守就一定是对的，但事实并不是如此
    当你所做的决定与和预期中的不符合时，就是你看错了，并没有任何理由，也不要对自己手中的股票抱有特殊地感情。一旦认识到错误就要毫不犹豫地止损。

  3. 第三个阶段就是听消息，当在哪里看到有人评论哪支股票的形态很好，就去研究那支股票，此时已经学到了一些东西了，也掌握了一些工具了，懂得了利用清单去对照。但是还是没有研究过基本面的情况下，就仓促的买进，到了最后因为股价没有按照预期的形态来走，就做了止损的决定。也没有再出现很大的亏损。

  4. 第四阶段慢慢地开始去了解股票的本质是什么，对于什么算低估，什么算高估有自己的判断，也更加有耐心了，能够持有现金，耐心等待买入时机的出现。同时也能够在自己持有股票在下跌的过程，因为对股票的耐心而不断加仓。而能够不断加仓的底气则来自对公司的确定性，成长性有了一定的认识。
